# Changelog

All notable changes to this repository are documented here. Dates are in ISO 8601 format (YYYY-MM-DD).

## Format

Each release entry lists:

- **Added** — new examples or features
- **Changed** — changes to existing examples (non-breaking)
- **Breaking** — changes that require manual action after update
- **Fixed** — bug fixes
- **Deprecated** — examples or features that will be removed
- **Removed** — removed examples or features
- **Security** — security fixes

Provider version compatibility is noted for each release.
